# Отчёт о качестве данных

## 1. Введение

**О датасете:**  
В работе анализируется датасет нобелевских лауреатов, содержащий информацию о всех лауреатах премии с 1901 по 2016 год. Датасет включает 911 записей и 23 признака: год присуждения, категория премии, имя лауреата, дата рождения, страна рождения, пол, организация, дата смерти, возраст на момент получения премии и другие.

**Предполагаемый заказчик:**  
Нобелевский фонд или исследовательская группа, изучающая историю науки и факторы, влияющие на присуждение премий. Также результаты могут быть интересны социологам и наукометристам.

**Возможные задачи ИАД:**
- Сегментация лауреатов по возрасту, полу, стране и категории
- Поиск аномалий (экстремально молодые или возрастные лауреаты)
- Описательная аналитика (динамика премий по годам, распределение по странам)

## 2. Паспорт датасета

**Размер датасета:** 911 строк, 23 столбца

**Основные признаки:**
- `year` — год присуждения премии
- `category` — категория премии (физика, химия, медицина, литература, мир, экономика)
- `full_name` — полное имя лауреата
- `birth_date` — дата рождения
- `birth_country` — страна рождения
- `sex` — пол
- `age` — возраст на момент получения премии
- `organization_name` — организация, в которой работал лауреат
- `death_date` — дата смерти (если есть)

## 3. Основные проблемы качества данных

| № | Тип проблемы | Описание | Потенциальный риск | Предлагаемое решение |
|---|--------------|----------|---------------------|----------------------|
| 1 | Мусорные данные | В столбцах `prize`, `prize_share`, `usa_born_winner`, `decade`, `female` содержатся иероглифы "宋栖" и "妊胰屠" | Невозможность анализа по этим признакам | Удалить столбцы или восстановить данные из оригинального источника |
| 2 | Некорректные значения | В столбце `age` присутствуют значения 0.0 | Искажение статистики среднего возраста | Удалить строки с возрастом 0 или заменить на NaN |
| 3 | Пропуски | В столбцах об организации (`organization_name`, `organization_city`, `organization_country`) пропущено более 50% данных | Ограниченный анализ по организациям | При анализе исключать пропуски или выделить их в отдельную категорию |
| 4 | Пропуски | В столбцах `death_date`, `death_city`, `death_country` ~30% пропусков | Некорректный расчёт продолжительности жизни | Учитывать только записи с полными данными для анализа продолжительности жизни |
| 5 | Грязные категории | В столбце `birth_country` встречаются исторические названия со скобками, например "Prussia (Poland)" | Затруднён анализ по современным странам | Создать новый столбец с извлечением современного названия страны |
| 6 | Грязные категории | В столбце `sex` присутствует мусорное значение "妊胰屠" вместо "Female" | Некорректный гендерный анализ | Заменить "妊胰屠" на NaN или восстановить пол из других источников |
| 7 | Выбросы | В столбце `age` есть экстремальные значения (17 лет, 90+ лет) | Не ошибка, но требует отдельного рассмотрения | Не удалять, но учитывать при интерпретации результатов |

## 4. План улучшения качества данных

### Приоритет 1 (критично)
1. **Очистка столбцов с мусором:** удалить или восстановить столбцы `prize`, `prize_share`, `usa_born_winner`, `decade`, `female`
2. **Исправление возраста:** удалить строки с возрастом 0
3. **Очистка пола:** заменить мусор в столбце `sex` на NaN или восстановить значения

### Приоритет 2 (важно)
4. **Обработка пропусков:** для анализа по организациям исключить строки с пропусками
5. **Создание новых признаков:** извлечь современные названия стран из столбца `birth_country`

### Приоритет 3 (можно отложить)
6. **Анализ выбросов:** исследовать экстремальные значения возраста как отдельные кейсы
7. **Заполнение пропусков в датах смерти:** для расчёта продолжительности жизни использовать только полные данные

## 5. Краткое резюме

В ходе работы были выявлены следующие ключевые проблемы:
- Значительная часть данных повреждена из-за ошибки кодировки (иероглифы в нескольких столбцах)
- Присутствуют пропуски в важных для анализа полях (организации, даты смерти)
- Обнаружены некорректные значения (возраст 0)
- Категориальные признаки содержат "грязные" значения (исторические названия стран, мусор в поле пол)

При дальнейшем анализе особого внимания требуют:
- Корректная обработка пропусков при расчёте статистик
- Учёт выбросов (молодые и возрастные лауреаты) как особых случаев
- Очистка географических данных для анализа по странам

Несмотря на выявленные проблемы, основные выводы по распределению премий по категориям и возрасту лауреатов можно считать надёжными, так как ключевые столбцы (`year`, `category`, `age`) содержат преимущественно корректные данные.
